Event Storming

Event Sourcing?

The authoritative source of truth is an
append-only stream of immutable events

  1. This happened
  2. then, this happened
  3. then, this happened
  4. ...

So, what happens?

The event is saved in our event store,
then broadcasted to every bounded context

✍️ A row for the news room (Editor)

👩‍💻 A row for readers (Consume)

📒 A row for internal reporting (Internal reporting)

🏞 A row for discovery (Discover)

💥 Invalidate the cache for that story
